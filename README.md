# ScriptScripter

Convert any text file into a script that automatically types it out as simulated keystrokes.

---

## Overview

ScriptScripter is a simple Windows Forms tool that reads a text file and generates a `.vbs` script which reproduces that text using simulated keyboard input.
Instead of typing manually, the generated script will "type" the contents of your file into whatever window is currently focused.

---

## How It Works

1. Select a text file
2. Choose a destination folder
3. Generate the script

The program will create a `.vbs` file named after your input file, containing a sequence of keystroke instructions.

---

## Features

* 📄 Converts any text file into a keystroke script
* 📂 Simple file + folder picker workflow (no manual typing required)
* ⏱ Configurable typing behavior (chunked input with delays)
* 🧹 Basic sanitization to prevent script-breaking characters
* 💬 Live preview of what will be generated

---

## Usage

1. Click **Pick Text File** and select your input file
2. Click **Pick Destination Folder**
3. Review the summary shown in the application
4. Click **Generate**

The program will create a file like:

```
yourfile_script.vbs
```

---

## Running the Output

* Double-click the generated `.vbs` file
* Focus the window where you want the text to appear QUICKLY
* The script will begin typing automatically after a 2 second delay

> ⚠️ The script sends keystrokes to the currently active window. Make sure the correct window is focused WITHIN TWO SECONDS of you running it.

---

## Notes

* Large files may take time to fully “type out”
* THIS RODEO HAS NO BRAKES. Understand the consequences of hitting the enter key before running a batch file generated by this program.
* Some special characters are replaced to avoid breaking the script
* Designed for simplicity rather than precision input control
